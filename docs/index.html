A modern CSS-like styling language with compile-time type checking.

Write modular, scoped styles with variables, mixins, nesting, themes, and if/else — then compile to clean CSS with zero runtime. Catch property errors before they reach the browser.

Features

Everything you need for modern styles

Adar combines the best ideas from CSS preprocessors with compile-time safety.

Type Checking

Detects invalid CSS property values at compile time. 150+ properties validated by type.

CSS Modules

Automatic class scoping with .modules.json mapping. No more conflicts.

Themes

theme name { } compiles to [data-theme="name"] selectors with CSS custom properties.

Mixins

Reusable style blocks with parameters. Keep your code DRY.

Nesting

CSS nesting with & parent reference. Compiles to clean nested CSS.

if/else

Compile-time conditionals. Toggle features on/off with variables.

CSS Variables

$var becomes var(--var) in output. Automatic scoping.

Property Validation

Wrong unit? Invalid color? Invalid property? Caught instantly at compile time.

Examples

See Adar in action

A button component with themes, variables, nesting, and mixins.

Adar
Compiled CSS
// Variables
$primary = #3b82f6
$radius = 8px

// Mixin
mixin card-base {
    border-radius: $radius
    padding: 16px
    box-shadow: 0 2px 8px rgba(0,0,0,0.1)
}

// Rule with nesting
.btn {
    include card-base
    background: $primary
    color: white
    border: none
    cursor: pointer

    &:hover {
        background: #2563eb
        transform: translateY(-2px)
    }

    &:active {
        transform: translateY(0)
    }
}
.btn__a1b2c3 {
  border-radius: var(--radius);
  padding: 16px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
  background: var(--primary);
  color: white;
  border: none;
  cursor: pointer;
}
.btn__a1b2c3:hover {
  background: #2563eb;
  transform: translateY(-2px);
}
.btn__a1b2c3:active {
  transform: translateY(0);
}
Install

Get started in seconds

Pilih cara instalasi yang paling mudah untuk kamu.

Windows — One-Click Install

Download dan install adar.exe + adarpc.exe langsung, tanpa perlu Python atau dependensi apapun.

Download adar.exe Download adarpc.exe
1

Download file .exe

Klik tombol di atas untuk download adar.exe dan adarpc.exe.

2

Install (tambah ke PATH)

Buka folder tempat file .exe disimpan, lalu jalankan PowerShell sebagai Administrator:

Set-ExecutionPolicy Unrestricted -Scope Process -Force; .\install.ps1

Atau double-click install.bat (auto-download .exe jika belum ada).

3

Selesai! Mulai pakai

Buka terminal baru, lalu coba:

adar build style.adar -o dist/
adar check style.adar
adarpc init my-project
adarpc build
adarpc serve
Cara alternatif: Python (pip)

Butuh Python 3.11+. Clone repo dan install dengan pip:

git clone https://github.com/zulsyam23-dot/adar.git
cd adar
pip install -e .

adar build src/style.adar -o dist/
adar check src/
adarpc init my-site
