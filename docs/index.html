A modern CSS-like styling language with compile-time type checking.

Features

Everything you need for modern styles

Adar combines the best ideas from CSS preprocessors with compile-time safety.

Type Checking

Detects invalid CSS property values at compile time. 150+ properties validated by type.

CSS Modules

Automatic class scoping with .modules.json mapping. No more conflicts.

Themes

theme name { } compiles to [data-theme="name"] selectors with CSS custom properties.

Mixins

Reusable style blocks with parameters. Keep your code DRY.

Nesting

CSS nesting with & parent reference. Compiles to clean nested CSS.

if/else

Compile-time conditionals. Toggle features on/off with variables.

CSS Variables

$var becomes var(--var) in output. Automatic scoping.

Property Validation

Wrong unit? Invalid color? Invalid property? Caught instantly at compile time.

Examples

See Adar in action

A button component with themes, variables, nesting, and mixins.

Adar
Compiled CSS
// Variables
$primary = #3b82f6
$radius = 8px

// Mixin
mixin card-base {
    border-radius: $radius
    padding: 16px
    box-shadow: 0 2px 8px rgba(0,0,0,0.1)
}

// Rule with nesting
.btn {
    include card-base
    background: $primary
    color: white
    border: none
    cursor: pointer

    &:hover {
        background: #2563eb
        transform: translateY(-2px)
    }

    &:active {
        transform: translateY(0)
    }
}
.btn__a1b2c3 {
  border-radius: var(--radius);
  padding: 16px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
  background: var(--primary);
  color: white;
  border: none;
  cursor: pointer;
}
.btn__a1b2c3:hover {
  background: #2563eb;
  transform: translateY(-2px);
}
.btn__a1b2c3:active {
  transform: translateY(0);
}

Install

Get started in seconds

Choose your preferred installation method.

Windows (Standalone .exe)

Download adar.exe and adarpc.exe from GitHub Releases. Run install.bat as Administrator to add to PATH.

adar build style.adar -o dist/
adar check style.adar
adarpc init my-project
adarpc build
adarpc serve

Python (pip)

git clone https://github.com/zulsyam23-dot/adar.git
cd adar
pip install -e .

adar build src/style.adar -o dist/
adar check src/
adarpc init my-site
