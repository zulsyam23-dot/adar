Publish Your Library

Follow these 3 steps to share your library with the community safely.

1

Verify your Code

Run adarpc publish in your project folder to ensure everything is bug-free.

2

Push to GitHub

Make sure your library code is available in a public GitHub repository.

3

Submit Entry

Submit a Pull Request to the AdarLib Registry with your repo link.
